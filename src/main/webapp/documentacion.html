Documentacion del servicio RESTful

Documentacion clientes

Obtener todos los clientes

'GET': http://localhost:8080/FVM/rest/clientes

Obtener un cliente por su ID

'GET': http://localhost:8080/FVM/rest/clientes/{id}

Registrar un nuevo cliente

'POST': http://localhost:8080/FVM/rest/clientes
Payload: {
    cuentaOrigen: int, 
    cuentaDestino: int,
    monto: float,
    tipoTransaccion: int,
    fecha: Date
}

Actualizar datos de cliente

'PUT': http://localhost:8080/FVM/rest/clientes/{idCliente}
Payload: {
    nombre: String, 
    direccion: String,
    usuario: String,
    password: String,
}

Eliminar un cliente

'DELETE': http://localhost:8080/FVM/rest/clientes/{idCliente}

Autenticar un cliente

'POST': http://localhost:8080/FVM/rest/clientes/login
Payload: {
    usuario: String, 
    password: String
}
